Simple Calculator

This is a basic calculator I made using Python. It runs in the terminal and does simple math operations.
What it can do:

    Add (+)

    Subtract (-)

    Multiply (*)

    Divide (/), with zero division check

How it works:

    It asks for two numbers and an operator

    Then shows the result

    You can keep using it until you choose to exit

How to run:
Just open the file and run:
python calculator.py


Note:
        Made for learning and practicing Python basics.
